小王子

然後他又回到狐狸那裡。

「再見了!」小王子 說。

「再見了!」狐狸 說。

「我的秘密很簡單:唯有心才能看得清楚,眼睛是看不到真正重要的東西的。」

小王子複述一遍:「真正的東西不是用眼睛可以看得到的。」



小王子留給我們與人生的六個關係練習
與自己的關係 每個人都有自己的孤單課題
你不需要變成你喜歡的大人
與愛人的關係 從心甘情願豢養與被豢養開始
因為他是你的玫瑰
與世界的關係 用「心」去尋覓你笑著的理由
分離的時候，學會道別
